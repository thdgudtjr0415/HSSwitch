# HSSwitch — 스피커·헤드셋·마이크를 한 번에 바꾸는 작은 트레이 앱

## 왜 만들었나

회의할 땐 헤드셋 마이크, 유튜브 볼 땐 스피커, 게임할 땐 다시 헤드셋 — 이런 식으로 오디오 장치를 하루에도 몇 번씩 바꿔야 하는 사람이라면 다들 공감할 텐데, Windows 기본 설정에서 재생 장치와 녹음 장치를 매번 따로 들어가서 바꾸는 건 은근히 번거롭습니다. 클릭 몇 번, 메뉴 몇 단계를 거쳐야 하니까요.

HSSwitch는 이 문제 하나만 해결하려고 만든 Windows 트레이 상주 프로그램입니다. 트레이 아이콘을 클릭하면 뜨는 작은 팝업에서, 혹은 미리 등록해둔 단축키 하나로 재생/녹음 장치 조합을 즉시 바꿀 수 있습니다.

- 개발 언어/도구: Python (pycaw, tkinter, pystray, pywebview)
- 배포 방식: GitHub Releases 기반 포터블 zip, 자체 자동 업데이트 내장
- 형태: 설치 프로그램이 아닌 포터블 프로그램, Windows 트레이에 상주

## 핵심 기능

- **개별 전환**: 메인 창에서 재생 장치(스피커/헤드셋)와 녹음 장치(마이크)를 목록에서 골라 즉시 전환
- **프로필**: "재생 장치 + 녹음 장치" 조합을 이름/아이콘/단축키와 함께 저장해두고 한 번에 전환
  - 메인 창 프로필 탭의 **"현재 장치로 빠른 추가"** 버튼을 누르면 지금 실제로 쓰고 있는 장치가 자동으로 채워진 채로 추가 창이 열립니다. 이름만 입력하면 끝.
  - 트레이 팝업에서도 **"+ 추가"**를 누르면 별도 창 없이 팝업 안에서 바로 입력 폼으로 바뀌고, 마찬가지로 지금 쓰는 장치가 자동으로 채워집니다.
  - 트레이 팝업의 프로필 타일을 **우클릭**하면 수정/삭제 메뉴가 떠서, 메인 창을 열지 않고도 관리할 수 있습니다.
- **글로벌 단축키**: 앱을 열지 않아도 등록해둔 단축키로 프로필을 바로 전환
- **트레이 팝업**: 트레이 아이콘 좌클릭 시 화면 우측 하단에 작은 팝업이 뜨고, 여기서 프로필/장치 선택은 물론 볼륨 조절과 음소거까지 가능합니다. 4.5초간 조작이 없으면 자동으로 닫힙니다.
- **장치 별칭**: Windows가 붙여준 복잡한 장치 이름 대신 원하는 이름으로 표시 (실제 장치 이름은 바뀌지 않습니다)
- **다크 모드**: 시스템 설정 따르기 / 라이트 / 다크 중 선택 (트레이 팝업에만 적용)
- **Windows 부팅 시 자동 실행**
- **자동 업데이트**: 실행할 때 새 버전이 있는지 확인해서 알림을 띄우고, 확인을 누르면 자동으로 받아서 교체 후 재시작합니다. 설정 탭 체크박스로 이 자동 확인을 켜고 끌 수 있고, 꺼두더라도 "업데이트 확인" 버튼으로 언제든 수동 확인이 가능합니다.
- **중복 실행 방지**: 여러 번 실행해도 창이 여러 개 뜨지 않고 기존 창/트레이 아이콘이 앞으로 나옵니다.

## 다운로드 및 설치

1. [최신 릴리즈 페이지](https://github.com/thdgudtjr0415/HSSwitch/releases/latest)에서 `HSSwitch.zip`을 내려받습니다.
2. 원하는 폴더에 압축을 풉니다. 설치 프로그램이 아니라 포터블이라 압축만 풀면 바로 쓸 수 있습니다.
3. 압축을 푼 폴더 안의 `HSSwitch.exe`를 실행합니다.

> **주의**: 압축을 푼 폴더 전체(`HSSwitch.exe`와 그 옆의 `HSSwitch_files` 폴더)를 그대로 유지해야 합니다. `HSSwitch.exe`만 따로 빼서 다른 곳으로 옮기면 실행되지 않습니다.

> **처음 실행할 때 "Windows의 PC 보호"(SmartScreen) 경고가 뜰 수 있습니다.** 코드에 문제가 있는 게 아니라 서명되지 않은 파일을 인터넷에서 받으면 Windows가 항상 띄우는 경고입니다. 창에서 **"추가 정보" → "실행"**을 누르면 정상적으로 실행됩니다.

이후로는 앱이 자동 업데이트를 처리하므로, 매번 이 페이지에 다시 와서 새로 받을 필요는 없습니다.

## 사용법

### 1) 트레이 팝업으로 빠르게 전환하기 (가장 많이 쓰게 될 방법)

화면 우측 하단 트레이 아이콘을 **좌클릭**하면 작은 팝업이 뜹니다.

- **프로필 타일**을 클릭하면 그 프로필에 저장된 재생+녹음 장치 조합으로 즉시 전환됩니다.
- **"+ 추가"** 타일을 누르면 팝업 안에서 바로 추가 폼으로 바뀝니다. 이름을 입력하고(재생/녹음 장치는 지금 쓰는 장치로 이미 선택되어 있습니다) 저장을 누르면 새 프로필이 만들어집니다.
- 프로필 타일을 **우클릭**하면 "수정" / "삭제" 메뉴가 뜹니다.
- 아래쪽 "재생 장치" / "녹음 장치" 목록에서 개별 장치를 클릭해도 바로 전환됩니다.
- 각 장치 목록 아래의 슬라이더로 볼륨을 조절하고, 스피커 아이콘을 누르면 음소거를 켜고 끌 수 있습니다.
- 4.5초 동안 아무 조작이 없으면 팝업은 자동으로 닫힙니다.

### 2) 메인 창

트레이 아이콘 **우클릭 → 열기**로 메인 창을 엽니다. 탭은 네 개입니다.

| 탭 | 용도 |
|---|---|
| 개별 전환 | 재생/녹음 장치를 각각 목록에서 골라 전환 |
| 프로필 | 프로필 추가/빠른 추가/수정/삭제/전환, 단축키 등록 |
| 장치 별칭 | 장치별로 원하는 표시 이름 지정 |
| 설정 | 테마, Windows 부팅 시 자동 실행, 자동 업데이트 확인 켜기/끄기, 수동 업데이트 확인 |

메인 창을 닫아도(X 버튼) 프로그램이 종료되는 게 아니라 트레이로 내려갑니다. 완전히 종료하려면 트레이 아이콘 우클릭 → 종료를 선택하세요.

### 3) 단축키로 전환하기

프로필 탭에서 프로필을 추가/수정할 때 "단축키" 칸에 `ctrl+alt+1` 같은 조합을 입력해두면, 앱 창을 열지 않고도 그 단축키만으로 프로필을 바로 전환할 수 있습니다. 관리자 권한으로 실행한 다른 프로그램이 포커스를 가진 상태에서 단축키가 안 먹는다면, HSSwitch도 관리자 권한으로 실행해보세요.

## 자주 묻는 질문

**Q. SmartScreen 경고가 떠요, 안전한가요?**
서명되지 않은 실행 파일을 인터넷에서 받으면 Windows가 항상 띄우는 표준 경고입니다. "추가 정보 → 실행"으로 넘기면 됩니다.

**Q. 업데이트는 어떻게 이루어지나요?**
실행할 때 GitHub에 올라간 최신 버전 정보를 확인해서, 새 버전이 있으면 알림을 띄웁니다. "예"를 누르면 새 버전을 내려받아 검증한 뒤 자동으로 교체하고 재시작합니다. 이 자동 확인 자체를 끄고 싶다면 설정 탭의 체크박스를 해제하면 됩니다.

**Q. 트레이 팝업이 안 떠요.**
트레이 팝업은 pywebview(내부적으로 Microsoft Edge WebView2)로 그려집니다. Windows 10/11엔 보통 기본 설치되어 있지만, 안 뜬다면 [WebView2 런타임](https://developer.microsoft.com/microsoft-edge/webview2/)을 설치해보세요.

**Q. 설정은 어디에 저장되나요?**
`%APPDATA%\HSSwitch\config.json`에 저장됩니다.
