# HSSwitch

스피커 / 헤드셋 / 마이크를 클릭 한 번, 단축키 한 번으로 빠르게 전환하는 Windows 트레이 프로그램입니다.

이 저장소는 HSSwitch의 **배포용 실행 파일과 업데이트 정보**를 올려두는 곳입니다.

## 다운로드

[최신 버전 다운로드](https://github.com/thdgudtjr0415/HSSwitch/releases/latest)

## 주요 기능

- **개별 전환** — 재생 장치(스피커/헤드셋)와 녹음 장치(마이크)를 목록에서 골라 즉시 전환
- **프로필** — 재생+녹음 장치 조합을 이름/아이콘/단축키와 함께 저장해서 한 번에 전환
- **글로벌 단축키** — 앱을 열지 않고도 등록한 단축키로 프로필 전환
- **트레이 팝업** — 트레이 아이콘 클릭 시 뜨는 작은 팝업에서 프로필/장치 선택 및 볼륨·음소거 조절
- **장치 별칭** — Windows가 붙인 복잡한 장치 이름 대신 원하는 이름으로 표시
- **다크 모드** — 시스템 설정 따르기 / 라이트 / 다크 선택 (트레이 팝업에 적용)
- **Windows 부팅 시 자동 실행**
- **자동 업데이트 확인** — 새 버전이 있으면 알려주고, 확인하면 바로 교체 및 재시작

## 설치

1. [최신 릴리즈](https://github.com/thdgudtjr0415/HSSwitch/releases/latest)에서 `HSSwitch.exe` 다운로드
2. 원하는 폴더에 두고 실행 (설치 프로그램 아님, 포터블)
3. 트레이 아이콘(파란 헤드셋)을 좌클릭하면 빠른 전환 팝업이, 우클릭하면 메뉴가 뜹니다.

## 제거

파일을 삭제하면 끝입니다. 자동 실행을 켜두었다면, 앱의 설정 탭에서 "Windows 부팅 시 자동 실행"을 먼저 꺼주세요.
